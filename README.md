# Algorithm Projects – C++

A collection of algorithm assignments implemented in C++ covering graph theory, sorting, minimum spanning trees, and algorithm design paradigms.

---

## Projects

### 1. Graph Project
Takes two undirected graphs as input and stores each in three different representations. Supports displaying edges from any chosen representation, and computes the intersection, union, subgraph relationship, highest-degree vertices, and equality between the two graphs.

### 2. Sorting Project
Implements multiple sorting algorithms and measures their execution time across various array sizes using randomly generated integers. Compares theoretical worst-case time complexity (Big-O) against actual runtime measured with `clock()`.

### 3. Kruskal Project
Implements Kruskal's algorithm using the Union-Find (Disjoint Sets) data structure on an undirected connected weighted graph. Outputs whether the graph is connected, the MST and its weight, set formation at each step, the parent array, cycle detection, and a path between two user-specified nodes.

### 4. ForTheFinal
Covers divide and conquer, greedy, and dynamic programming problems including: finding the maximum element and sum of an array recursively, minimizing average file waiting time, maximizing song popularity within a storage limit, and finding the minimum travel cost through cities using DP.
